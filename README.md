# Titanic Hayatta Kalma Tahmini (Makine Öğrenmesi)

Bu proje, Kaggle üzerindeki meşhur Titanic veri setini kullanarak yolcuların özelliklerine göre hayatta kalıp kalmayacaklarını tahmin eder.

## 🚀 Projenin Amacı
Bu çalışma, Lojistik Regresyon algoritması ile bir sınıflandırma modeli oluşturmayı ve veri analizini yapmayı amaçlar.

## 📊 Kullanılan Teknolojiler
* Kaggle Titanic Dataset
* Python (Pandas, Seaborn, Scikit-learn)

## 🛠️ Uygulanan Adımlar
Veri ön işleme, eğitim (%80) ve test (%20) ayrımı yapılarak model eğitilmiştir.

## 📈 Sonuçlar
Modelin doğruluk oranı (Accuracy) yaklaşık **%73** olarak hesaplanmıştır.

## 💡 Gelecek Çalışmalar
Geliştirme için Random Forest veya XGBoost gibi daha karmaşık algoritmalar denenebilir.
